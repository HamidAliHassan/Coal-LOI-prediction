# Coal-LOI-prediction
Loss on Ignition (LOI%) serves as a critical proxy for combustion efficiency and unburnt carbon retention in coal-utilizing industries. This study presents a predictive machine learning framework designed to map non-linear interactions between coal proximate parameters—Total Moisture (TM), Ash content (ASH), Volatile Matter (VM), Fixed Carbon (FC), Sulfur (S), and Gross Calorific Value (GCV)—and the resulting LOI%.A comparative evaluation across three tree-based ensemble architectures demonstrated that the Gradient Boosting Regressor achieved superior predictive accuracy, yielding an R^2 score of 0.9976 and a Root Mean Squared Error (RMSE) of 0.00012, outperforming Random Forest (R^2 = 0.9913) and tuned XGBoost (R^2 = 0.9814.Under severe operational target bounds constraining LOI% to < 1.0% (< 0.01 in decimal scale), Monte Carlo matrix simulations containing 50,000 synthetic permutations revealed that a sub-1% LOI target is strictly unreachable under current physical raw material spectrum constraints. Optimization boundaries identified a strict physical floor for LOI% at ~1.625%. The optimal mineral profile to minimize unburnt potential requires maximizing Total Moisture (~9.86%), maintaining Ash content around 15.9%, balancing Volatile Matter (~33.4%), and severely suppressing Sulfur content (< 0.0085%).
